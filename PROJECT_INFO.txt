【快速开始】

1. 验证环境:
   ./verify.sh

2. 运行系统:
   ./run.sh
   或
   java -jar target/ontology-dynamic-network-1.0.0.jar

3. 基本操作:
   ontology> start                              # 启动引擎
   ontology> load ontology config/ontology.yml  # 加载本体
   ontology> load actions config/actions.yml    # 加载动作
   ontology> load rules config/rules.yml        # 加载规则
   ontology> demo                               # 运行演示
   ontology> help                               # 查看帮助
   ontology> exit                               # 退出

【技术栈】
- Java 17+
- Maven 3.6+

- SLF4J + Logback (日志)

【核心特性】
1. 事件驱动架构 - 响应式状态变更处理
2. 声明式配置 - 通过配置文件定义业务逻辑
3. 语义化抽象 - 统一的本体模型
4. 高度可扩展 - 插件式设计
5. 图谱化管理 - 对象关系网络

【命令参考】
start            启动动力学决策网络引擎
stop             停止引擎
load <type> <file>   加载配置文件 (ontology/actions/rules)
ingest <type>    摄入外部数据 (git_commit/requirement/defect)
update <id> <attr> <val>   更新对象属性(触发规则)
status           显示引擎状态
objects          显示所有本体对象
rules            显示所有规则
demo             运行演示场景
help             显示帮助信息
exit/quit        退出系统
